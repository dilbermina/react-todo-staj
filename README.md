# React Todo App

##  Proje Hakkında
Bu proje, React ve Bootstrap kullanılarak geliştirilmiş basit bir görev takip (Todo) uygulamasıdır.

## Özellikler
- Görev ekleme
- Görev listeleme
- Görev güncelleme
- Görev silme

##  Kullanılan Teknolojiler
- React
- TypeScript
- Bootstrap
- Vite
